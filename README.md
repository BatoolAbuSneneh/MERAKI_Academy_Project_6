A website that delivers halls and user can publish or reserve a hall for a wedding

## 🏁 Getting Started <a name = "getting_started"></a>

clone Repostiory to your local machine

```
https://github.com/CodeWarriorsTeam/MERAKI_Academy_Project_6.git
```

in Backend gitbash
```
npm i 
npm i mysql2
```

in Frontend gitbash
```
npm i 
npm i axios
```

then in backend gitbash
```
npm run dev
```
and in frontend gitbash 
```
npm start
```

install mysql
```
mysql DB
```

### Installing

if package json isn't installed:

```
npm init -y 
```
